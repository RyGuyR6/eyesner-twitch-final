EYESNER LIGHTNING STORM V2 OBS PACKAGE

INSTALLATION
1. Extract this ZIP to a normal folder on your PC.
2. Close OBS Studio if it is running.
3. Double-click Install-Eyesner.bat.
4. Wait for the installer to copy the package into your Videos folder and create the OBS scene collection.
5. Open OBS Studio and switch to the "Eyesner Lightning Storm V2" scene collection.

WHAT THE INSTALLER DOES
- Detects your OBS Studio configuration folder automatically.
- Preserves your existing OBS profiles and scene collections.
- Creates a new scene collection named "Eyesner Lightning Storm V2".
- Installs these scenes automatically:
  * Starting Soon
  * Intermission
  * Be Right Back
  * Stream Ending
  * Gameplay Overlay
  * Follower Alert
  * Subscriber Alert
- Copies the included media files into your Videos folder for OBS to use.
- Includes LightningStinger.webm in the installed package for transition setup.

NOTES
- If a collection with the same name already exists, the installer stops instead of overwriting it.
- Run the BAT launcher if Windows blocks direct PowerShell script execution.
